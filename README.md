# Football Career Analysis using Python

This project focuses on analyzing the career statistics and performance trends of a renowned football player using Python and Exploratory Data Analysis (EDA).

## Project Objective
The objective of this project is to extract meaningful insights from sports data through data cleaning, preprocessing, visualization, and statistical analysis.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Tasks Performed
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Performance Trend Visualization
- Career Statistics Analysis
- Insight Generation from Sports Data

- ## 📂 Project Files

| File Name | Description |
|-----------|-------------|
| `EDA on Career Analysis.ipynb` | Main analysis notebook |
| `EDA on Football Matches (1).ipynb` | Match-level exploratory analysis |
| `football_analysis_dataset.csv` | Dataset used for analysis |
| `Career Analysis of Cristiano Ronaldo.pptx` | Presentation containing project insights |

---

## 📊 Key Insights

- Player performance improved significantly over the years
- Goal-scoring consistency increased during peak seasons
- Certain tournaments contributed heavily to total goals
- Performance trends revealed career growth patterns
- Visual analysis helped identify high-performing periods

This project helped strengthen my analytical thinking, data visualization, and problem-solving skills using real-world sports datasets.
