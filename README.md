# Football Career Analysis using Python

This project focuses on analyzing the career statistics and performance trends of a renowned football player using Python and Exploratory Data Analysis (EDA).

## Project Objective
The objective of this project is to extract meaningful insights from sports data through data cleaning, preprocessing, visualization, and statistical analysis.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Tasks Performed
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Performance Trend Visualization
- Career Statistics Analysis
- Insight Generation from Sports Data

## Key Insights
- Analyzed player performance trends over seasons
- Identified statistical patterns and consistency
- Visualized career growth using charts and graphs

## Project Outcome
This project helped strengthen my analytical thinking, data visualization, and problem-solving skills using real-world sports datasets.
